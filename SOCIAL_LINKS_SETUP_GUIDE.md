## Troubleshooting

### "My links aren't saving!"
**Solution:**
1. Clear browser cache (Ctrl+Shift+Delete)
2. Refresh page (F5)
3. Try adding link again
4. Check you're signed in (look for "Ï€ Auth" badge)

### "I only see 1 link field but I have a Basic plan!"
**Solution:**
- Free plan shows only Twitter field
- Basic plan shows Twitter, Instagram, YouTube
- To see more: [Upgrade to Premium](http://droplink.space/subscription)

### "I added a custom link but it's not showing!"
**Solution:**
- Custom links only available for **Premium/Pro** users
- Free and Basic plans: Use standard platforms only

### "The page says 'You have reached your plan's social link limit'"
**Solution:**
1. Check your current plan (Crown icon in header)
2. You've hit your plan's limit
3. Upgrade to add more links
4. Or remove a link to add another one
