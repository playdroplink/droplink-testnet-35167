## Overview
DropLink now supports animated GIF backgrounds for your storefront, similar to popular link-in-bio platforms like Linktree. This feature allows you to create dynamic, engaging profile pages that capture your brand's personality.

## How to Add GIF Backgrounds

### **Step 1: Access Design Settings**
1. Open your DropLink Dashboard
2. Navigate to the **Design** tab
3. Choose from **Ready-to-Use Themes** or scroll down for custom options

### **Step 2: Enable GIF Background**
1. Select the **"GIF Background"** radio button
2. The GIF configuration panel will appear

### **Step 3: Choose Your GIF**

#### **Option A: Use Ready-to-Use Themes** ‚≠ê *Recommended*
- Select themes with built-in GIF backgrounds:
  - **Startup Vibe**: Purple gradient animation
  - **Music Maestro**: Pink wave animations  
  - **Cyberpunk Code**: Neon cyberpunk effects
  - **eSports Pro**: Gaming effects
  - **NFT Creator**: Colorful animations
- One-click setup with professional designs

#### **Option B: Upload Your Own GIF**
1. Click **"Choose GIF File"** button
2. Select a GIF file from your device (max 10MB)
3. File will be instantly uploaded and previewed
4. No need to host the file elsewhere!

#### **Option B: Use Sample Backgrounds**
- Choose from 3 pre-selected animated backgrounds:
  - **Gradient Flow**: Smooth color transitions
  - **Abstract Waves**: Dynamic wave patterns  
  - **Cosmic**: Space-themed animation

#### **Option C: Use Custom GIF URL**
1. Enter your GIF URL in the text field
2. Must be a direct link to a `.gif` file
3. Example: `https://example.com/your-background.gif`

### **Step 4: Preview & Save**
1. Preview your GIF in the preview window
2. Check the mobile preview on the right
3. Click **"Save Changes"** to apply

## FAQ

**Q: Can I upload GIF files directly?**
A: Yes! You can now upload GIF files directly from your device (up to 10MB).

**Q: What's better - uploading or using URLs?**
A: Uploading is more reliable since the file is stored with your profile. URLs depend on external sources.

**Q: Are GIF backgrounds free?**
A: Yes! Both upload and URL options are included in all plans, including the free tier.

**Q: Will GIFs slow down my page?**
A: Properly optimized GIFs (under 5MB) shouldn't significantly impact loading times.

**Q: Do GIF backgrounds work on mobile?**
A: Yes! GIF backgrounds are fully optimized for mobile devices.

**Q: Can I disable GIFs for certain visitors?**
A: This feature respects user "reduced motion" preferences automatically.
